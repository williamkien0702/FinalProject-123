# Play Mode scenario life cycle

## Active Scenario Window

The Active Scenario window displays the status of all the instances in your scenario. You can also access certain aspects like free running instances from this window. For more details on configuration options, refer to the [Scenario configurations page](play-mode-scenario-configurations.md).

### Check the status of all instances

The Active Scenario window displays the following information for each Play Mode Scenario:

* Status
* Port
* Address
* Errors

To open the Active Scenario window:

1. Enter Play Mode.
2. Expand the Play Mode status dropdown.
3. Select the status window icon.
